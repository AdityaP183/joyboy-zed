# JoyBoy Theme for [Zed](https://zed.dev)

> A custom dark theme and icon pack for [Zed](https://zed.dev), inspired by iconic themes with a unique JoyBoy twist.

## About

**JoyBoy Zed** is a personal theme crafted to enhance the visual experience of the Zed editor. It blends aesthetics and usability, focusing on a clean dark interface with expressive colors and cohesive icons.

The theme draws inspiration from:

* [Dracula](https://github.com/dracula/zed)
* [Dark Magic Themes](https://github.com/davidsmorais/dark-magic-themes)

The icon set is inspired by:

* [Charmmono Theme](https://github.com/sumangal44/charmmono-theme)

All assets have been customized and refined to create a consistent and distinctive look.
